# 🔐 Razorpay Real Payment Integration Setup

This movie booking application now integrates with **Razorpay**, India's leading payment gateway, for real and secure UPI payments.

## Step 1: Get Razorpay Account & API Keys

### 1.1 Create Razorpay Account
1. Go to https://razorpay.com
2. Click "Sign Up" and create account (or login if you have one)
3. Complete KYC verification (1-2 business days)

### 1.2 Get Your API Keys
1. Go to Dashboard → Settings → API Keys
2. You'll see two modes:
   - **Test Mode** (for development) - Use playground money
   - **Live Mode** (for production) - Real payments

**For Development (Recommended):**
- Use Test Mode keys
- No real money is charged
- Perfect for testing payment flow

**For Production:**
- Switch to Live Mode
- Real money will be processed
- Configure webhook for payment confirmations

---

## Step 2: Configure Backend with API Keys

### 2.1 Create `.env` File
Create a `.env` file in the `backend/` folder:

```env
# Razorpay Payment Gateway
RAZORPAY_KEY_ID=rzp_test_YOUR_KEY_ID
RAZORPAY_KEY_SECRET=YOUR_KEY_SECRET

# Database configuration (if not already set)
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=gaurav
DB_NAME=moviebook_db

# JWT
JWT_SECRET=yash_gaurav_movie_booking_project_secret_key
```

### 2.2 Where to Get Keys
1. In Razorpay Dashboard → Settings → API Keys
2. You'll see:
   - Key ID (starts with `rzp_test_` or `rzp_live_`)
   - Key Secret (long string)
3. Copy both and paste in `.env` file

**Example:**
```env
RAZORPAY_KEY_ID=rzp_test_1234567890abc
RAZORPAY_KEY_SECRET=xxxxxxxxxxxxxxxxxxx
```

---

## Step 3: Restart Backend Server

After adding `.env` file with Razorpay keys:

```bash
cd backend
npm install dotenv  # If not already installed
node server.js
```

The backend will load your Razorpay keys from `.env` file.

## How Real Payments Work

### Payment Flow:
```
1. User selects seats → Backend reserves seats
   ↓
2. User goes to Payment Page
   ↓
3. User clicks "Pay with Razorpay" button
   ↓
4. Razorpay checkout opens → Shows multiple UPI apps (GPay, PhonePe, Paytm)
   ↓
5. User selects UPI app and completes payment
   ↓
6. Razorpay returns payment confirmation
   ↓
7. Backend verifies payment signature (security check)
   ↓
8. Payment amount verified matches booking amount exactly
   ↓
9. ✅ Booking confirmed → Seats locked
10. 📧 Receipt sent to email
```

## Testing Payment Scenarios

### Test Card Details for Razorpay (Test Mode Only)
When Razorpay checkout opens in Test Mode, you can use:

**Card Payments:**
- Card Number: `4111 1111 1111 1111`
- Expiry: Any future date
- CVV: Any 3 digits
- Result: Payment succeeds

**For UPI Simulation:**
- Select any UPI app in Razorpay checkout
- Razorpay provides test UPI IDs automatically

### What Generates Real UTR Numbers?
Real UTR numbers are generated by Razorpay when:
1. User completes payment in Razorpay checkout
2. Payment gateway processes the transaction
3. Transaction ID (UTR) is returned to your app
4. Backend stores it in payment record

## Key API Endpoints

### Create Payment Order
```
POST /api/bookings/{bookingDbId}/create-order

Response:
{
  "success": true,
  "data": {
    "orderId": "order_xxxxx",
    "amount": 450.00,
    "currency": "INR",
    "bookingId": "BK20260414001",
    "keyId": "rzp_test_xxxxx"
  }

```

### Verify Payment
```
POST /api/bookings/{bookingDbId}/verify-payment

Body:
{
  "orderId": "order_xxxxx",
  "paymentId": "pay_xxxxx",
  "signature": "xxxxxxxx"
}

Response (Success):
{
  "success": true,
  "data": {
    "bookingId": "BK20260414001",
    "transactionId": "pay_xxxxx",
    "utr": "12345678901",
    "totalPaid": 450.00,
    "seats": ["A1", "A2", "A3"],
    "verifiedAt": "2026-04-14T10:30:45Z"
  }
}
```

## Troubleshooting

### Issue: "Invalid Razorpay API Keys"
**Solution:** Check `.env` file has correct keys without extra spaces

### Issue: "Payment verification failed"
**Solution:** Ensure Key Secret is correct - check Razorpay Dashboard again

### Issue: "Amount mismatch"
**Solution:** This is a security feature - payment must match booking amount exactly

### Issue: Razorpay checkout doesn't open
**Solution:** 
1. Check browser console for errors
2. Verify Razorpay script loaded: <script src="https://checkout.razorpay.com/v1/checkout.js"></script>
3. Check Key ID is valid
